Section 1: Overview

Originator:Minnesota DNR - Division of Forestry

Title: MNDNR Forest Inventory Old Growth

Abstract: This layer is a digital inventory of currently designated or candidate old growth forest stands on lands administered by the Department of Natural Resources (DNR). This data originates from the Inventory Module of the Minnesota DNR's 4Trees System. Old growth forest designations are determined through an interdisciplinary evaluation process by DNR staff in divisions of Forestry, Fish and Wildlife, Ecological and Water Resources, and Parks and Trails. These data are updated on a continuous basis, as needed.

Also included is the old growth Special Management Zone (SMZ), a 330-foot buffer around the Old Growth layer.

Purpose: Forest resource planning, harvest plans, treatment plans, wildlife habitat assessment, natural heritage review, native plant community mapping support, historical vegetation studies.

Section 2: Data Quality

Attribute Accuracy: Data are in use by DNR staff on a daily basis. Attributes are fixed as problems are made known

Logical Consistency:

Completeness: As new lands are acquired, 4Trees Stand Inventory data collection is scheduled.

Horizontal Positional Accuracy: Features are digitized onscreen over standard 24k digital raster graphics (DRGs) and U.S. Geological Survey (USGS) digital orthophoto quadrangles (DOQs) and will inherit the horizontal positional accuracy of those products.

Vertical Positional Accuracy: Not Applicable

Lineage: Original source layer was MNDNR's enterprise geodatabase: Forest Inventory Managment System (FIM). Data are now extracted on a regular basis from the MNDNR's 4Trees Inventory Module (LRM)

Section 5: Attributes

Overview: Stand Key: Unique system-generated number to identify the stand

SLabel: Stand label consisting of the stand's FIM_type and Cover Type Classification/Size/Density

Cover Type Code: Cover type code (see table below)

Site Index: Site Index Value (a measure of site productivity)

Main Species Code: Code for species used for the site index value (see table below)

Old Growth Status: Text field indicating the current old growth designation status (designated, designated FOG, or candidate)(see table below)

Old Growth ID: Unique ID for each old growth site (which can be comprised of multiple stands)

Old Growth Comment: General notes section that sometimes includes info about OG

Location: t-number; unique for each stand

RAN: DNR Division of Forestry Administrative Area Code (see table below)

County Number: Number assigned to the county name (see table below)

PLS: PLS value

Stand Age Last Inventory: Computed from most recent inventory activity

Stand Acres: Total acres in the stand

Stand Age - Current: Current stand age (Current Year - Last Inventory Year) + Last Inventory Age
